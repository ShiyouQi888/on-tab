# 更新日志 (CHANGELOG)

## [1.1.0] - 2025-12-30

### 修复 (Fixed)
- **国际化 (i18n)**: 修复了设置页面中的翻译键错误和翻译不完整的问题。
- **构建稳定性**: 修复了 Supabase 同步服务的 TypeScript 类型错误，解决了 Vercel 构建失败的问题。
- **设置模块**: 修复了分类删除确认框中的翻译变量作用域问题。

### 优化 (Improved)
- **数据同步**: 优化了书签和分类的同步逻辑，增强了数据一致性和错误处理。
- **代码质量**: 为同步服务和迁移服务添加了显式的接口定义，提升了代码的可维护性。
- **界面体验**: 完善了所有弹窗和通知的中英文翻译支持。

### 更新 (Updated)
- 插件版本号更新至 1.1.0 (Stable)。
